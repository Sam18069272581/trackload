# trackload
Demo: https://sam18069272581.github.io/trackload/

TrackLoad is a web application for recording, tracking, and visualizing tasks, workloads, or resources. It is designed for users and contributors of all genders and backgrounds. The project aims for an approachable structure, easy onboarding, and straightforward extensibility.

Features
Create, edit, and delete loads/tasks
Status tracking (New, In Progress, Completed)
Filtering, sorting, and search
Summary statistics (Total, Open, Completed)
Local storage (offline support) with export options (CSV/JSON)
Simple setup for contributors: clear code structure and well-defined interfaces

Tech Stack
HTML
Local storage (localStorage) for offline support
